# MR.COFFEE - Responsive Coffee Shop Website
MR.COFFEE is a responsive coffee shop website developed as a Human and Computer Interaction (HCI) project. The website was designed to provide an engaging digital experience for coffee lovers while maintaining usability, accessibility, and responsive design principles across desktop, tablet, and mobile devices.

The project includes five core pages: Home, Menu, Order, Rewards, and About Us. Users can browse featured products, explore promotions, learn about the company, view categorized menu items, and place orders through an interactive order form with JavaScript validation.

# Features
1. Responsive design for desktop, tablet, and mobile devices
2. Interactive Home page with featured products and promotional content
3. Categorized Menu page with product filtering functionality
4. Rewards page showcasing promotions and mobile app incentives
5. About Us page featuring company history, mission, vision, and founder story
6. Order page with dynamic form validation using JavaScript
7. Consistent UI/UX design following Human-Computer Interaction principles
8. External CSS styling and organized project structure

# Technologies Used
1. HTML5
2. CSS3
3. JavaScript
4. Figma
5. Responsive Web Design
   
# Key Functionalities
1. Product category filtering
2. Interactive navigation between pages
3. Form validation without regular expressions
4. Responsive layouts using media queries
5. Dynamic user interactions with JavaScript
